Welcome to Planes API documentation

Planes Routes

Method URL Body Description
GET /planes Get a list of all the plane, will present 10 results per page
GET
  1. /planes?page=2
  2. /planes?perPage=4
  3. /planes?reverse=yes
  4. /planes?sort=year
    5. Note: You can use the above for all GET methods
     Note: You can also combined the above for multi filter option
     (/planes?sort=year&reverse=yes&perPage=3)
  1. Get a list of all the plane, will present 10 results in page 2
  2. Get a list of all the plane, will present 4 results in page
  3. Get a list of all the plane, will present 10 results in page in reverse order (default sort=year
  4. Get a list of all the plane, will present 10 results in page, Defines what the list will be sorted by
GET /planes/search?s={keyword(Name, Manufacturer, Info)}
(/planes/search?s=F-16)		 search for a plane or a plane description/info, will present 10 results per page
GET /planes/category/{category name}
(/planes/category/fighter)		 Get a list of all the planes in selected category, will present 10 results per page
GET /planes/year?min={value}&max={value}
(/planes/year?min=1945&max=2001)		 Get a list of all the planes that manufactured between 1945 and 2001, will present 10 results per page (you can add only minimum year or maximum as well
/planes/year?min=1945)

{

 "info": "model description ,
 "year": yyyy,

Update a plane in the data base, You need a token for this query (only admin users alowed to modified other users entry)
DELETE /planes/{_id} Delete a plane from the data base, You need a token for this query (only admin users alowed to delete other users entry)

Users Routes

Method URL Body Description
GET
  1. /users?page=2
  2. /users?perPage=4
  3. /users?reverse= yes
  4. /users?sort=user id
    5. Note: You can use the above for all GET methods
     Note: You can also combined the above for multi filter option
  1. Get a list of all the users, will present 10 results in page 2
  2. Get a list of all the users, will present 4 results in page
  3. Get a list of all the users, will present 10 results in page in reverse order (default sort=user id
  4. Get a list of all the users, will present 10 results in page, Defines what the list will be sorted by
GET /users Get a list of all the users, will present 10 results per page, You need a token for this query (only for admins)
GET /users/info Get a user info, You need a token for this query (only for corrent user)

Delete /users/{_id}
